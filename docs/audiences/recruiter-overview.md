# Recruiter Overview

Helix Platform is a production-ready, GitOps-first platform designed for multi-cluster operations. Key selling points:

- Immutable infrastructure (Talos)
- GitOps orchestration (ArgoCD)
- Developer-friendly pipelines (Devtron)
- Zero-trust networking (NetBird)
- Autonomous operations (Iron Spider)
- Observability-first (Prometheus + Grafana)

Candidate talking points:

- Built for scale and resilience
- Emphasis on automation and SRE workflows
- Clear separation of control and data planes
